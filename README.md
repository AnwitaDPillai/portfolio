# Anwita D Pillai — Personal Portfolio

The open-source codebase for my personal digital home.

**Live Deployment:** [anwitadpillai.github.io/portfolio](https://AnwitaDPillai.github.io/portfolio/)

---

## Design & Architecture

Unlike traditional single-column web builds, this platform utilizes an asymmetric magazine layout engine to frame my text alongside an integrated academic accolades ledger.

* **Typography:** Set in classic `Cinzel` for stark archival classicism, paired with premium `Playfair Display` serif body prose and high-contrast `Plus Jakarta Sans` for metadata micro-tags.
* **Aesthetic:** Rooted in an ivory/book-leaf paper color story directly synchronized with my writing brand ecosystem.
* **Development:** Hand-coded from scratch using responsive semantic HTML5, fluid layout grids, and performance-optimized CSS3 architecture. Zero overhead, zero heavy web-builder dependencies.
